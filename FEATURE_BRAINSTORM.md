# MacroManage - Feature Brainstorm & Roadmap

## ✅ Recently Implemented (v3.0)
1. **Real Weather API** - Live forecasts using Open-Meteo
2. **Notification System** - Bell icon with badge count, notification panel
3. **Interactive Maps** - Event creation preview & details view
4. **Comprehensive Event Details** - Full stats page with availability, expenses, carpool
5. **Friend Groups** - Save and reuse friend lists
6. **Dark Mode** - Theme persistence
7. **Search Events** - Filter by title, location, friends
8. **Edit Events** - Modify after creation

## 🚀 New Feature Ideas

### **1. Recurring Events**
- Weekly/monthly event templates
- Auto-create future instances
- Bulk invite management
- Example: "Weekly Basketball - Every Saturday 3pm"

### **2. Event Templates**
- Save event configurations as templates
- Pre-filled: friends, location, budget, times
- Quick create from template
- Example: "Monthly Dinner Club" template

### **3. Availability Calendar Integration**
- Sync with Google Calendar, Apple Calendar
- Auto-block busy times
- Show free/busy status to friends
- Smart scheduling suggestions

### **4. Split Payment Integration**
- Venmo/PayPal/Zelle links
- Track who paid, who owes
- Send payment reminders
- Settlement suggestions

### **5. RSVP Deadline & Reminders**
- Set response deadline
- Auto-remind non-responders
- Escalating reminder frequency
- Final headcount notification

### **6. Event Photos & Memories**
- Upload event photos
- Shared photo album
- Post-event recap
- Memory timeline

### **7. Dietary Restrictions & Preferences**
- Save friend dietary info
- Restaurant suggestions based on group needs
- Allergy alerts
- Menu planning assistance

### **8. Transportation Optimization**
- Optimal carpool routes
- Uber/Lyft cost estimates
- Public transit directions
- Parking availability

### **9. Event Polls & Voting**
- Vote on multiple aspects: date, time, location, activity
- Ranked choice voting
- Real-time results
- Auto-confirm based on majority

### **10. Budget Tracking & Analytics**
- Per-event budget vs actual
- Monthly spending reports
- Cost per person trends
- Savings suggestions

### **11. Friend Availability Patterns**
- Learn when friends are typically free
- Smart scheduling based on patterns
- Conflict prediction
- Best time suggestions

### **12. Event Waitlist**
- Max capacity setting
- Waitlist management
- Auto-invite from waitlist
- Priority ordering

### **13. Weather-Based Suggestions**
- Indoor/outdoor activity recommendations
- Backup plans for bad weather
- Seasonal activity ideas
- Weather alerts 24h before

### **14. Group Chat Integration**
- In-app messaging per event
- WhatsApp/Telegram integration
- Announcement broadcasts
- Quick polls in chat

### **15. Event Check-In**
- QR code check-in
- Attendance tracking
- Late arrival notifications
- No-show tracking

### **16. Collaborative Planning**
- Co-hosts with edit permissions
- Task assignments (bring drinks, setup, etc.)
- Checklist management
- Responsibility tracking

### **17. Event Ratings & Reviews**
- Rate venues/restaurants
- Friend group compatibility scores
- Activity success metrics
- Improvement suggestions

### **18. Smart Notifications**
- Customizable notification preferences
- Digest mode (daily summary)
- Priority levels
- Do Not Disturb scheduling

### **19. Event Series & Seasons**
- Group related events (Summer BBQ Series)
- Season statistics
- Tradition tracking
- Anniversary reminders

### **20. Social Features**
- Public events (open to friends-of-friends)
- Event discovery
- Trending activities
- Community recommendations

5. **Dietary Restrictions** - Important for food events

6. **Calendar Integration** - Technical complexity but high value
7. **Event Photos** - Engagement feature

10. **Collaborative Planning** - Enables larger events

### **Low Priority** (Nice-to-Have)
